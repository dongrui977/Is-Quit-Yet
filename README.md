# 辞了吗 (QuitYet) 
> “只要我不辞职，老板就永远过不上他想要的生活。”

### 🛠️ 项目初衷
本产品用于监控当代打工人的生命体征。每天确认一次：今天还没离职吗？
- **如果你还在苟活**：请点击打卡，换上“打工牛马”专属头像。
- **如果你已解脱**：请点击爆破，头像将永久进化为“赛博自由人”。

### 🎭 状态演化逻辑
| 状态 | 头像 | 含义 |
| :--- | :--- | :--- |
| Day 1 | 😐 | 初入职场，面无表情 |
| Day 7 | 🧟 | 尸化进行中，靠咖啡吊命 |

| Day 365| 🗿 | 已成化石，与工位融为一体 |

### 🚀 快速启动 (Vibe Mode)
1. 每天早上 9 点准时同步你的怨气。
2. 连续 3 个月不打卡，系统自动判定你已“在工位圆寂”。
